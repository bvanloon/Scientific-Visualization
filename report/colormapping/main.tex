\chapter{Color Mapping} % (fold)
\label{cha:color_mapping}

Color mapping is widely used for the visualization of scalar data. This is done by associating a color with every scalar value that is visualized. In this chapter we will discuss the colormaps included in the application. \Cref{s:colormapping:introduction} highlights the important features of colormaps and discusses several color maps that are implemented. The next section discusses how we have implemented our color maps. Finally, \cref{s:colormapping:results} shows the results of applying color maps to our data set.
